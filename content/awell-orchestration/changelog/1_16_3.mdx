## Added

### startHostedPathwaySession mutation

Mutation added to start a hosted pathway session.

### stopTrack mutation

Mutation added which allows to stop a track of an ongoing pathway.

### Webhooks

Five new webhooks have been added to support the hosted pathway & activity integration:

- pathway.started
- pathway.completed
- session.started
- session.completed
- session.expired

More information on these webhooks can be found under the "Webhooks" section.
