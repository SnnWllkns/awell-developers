---
title: Get EMR Report Details
description: Get the body of an EMR report activity
---

This request allows us to fetch the details of an EMR report activity.

## Request

### Query

```graphql
query GetEmrReport($id: String!) {
  emrReport(id: $id) {
    report {
      id
      message_html
      metadata {
        id
        label
        value
      }
    }
  }
}
```

### Variables

An example `id` for an EMR report is "9rjfb9qE4pEY".

```json
{
  "id": "{{EMR_REPORT_ID}}"
}
```

<br />

#### Where to find the EMR report id

The id of the EMR report can be found in the `object.id` field of an activity.

```json
{
  "activity": {
    "id": "{{ACTIVITY_ID}}",
    "object": {
      "id": "{{EMR_REPORT_ID}}",
      "type": "emr_report"
    },
    ...
  }
}
```
