---
title: Get published pathway definitions
description: Get all published pathway definitions in your organization
---

Gets the full list of published pathway definitions in your tenant or organization.

## Request

### Query

```graphql
query GetPublishedPathwayDefinitions {
  publishedPathwayDefinitions {
    publishedPathwayDefinitions {
      id
      title
      version
      dataPointDefinitions {
        id
        key
        category
        valueType
        optional
        range {
          min
          max
        }
        unit
        possibleValues {
          label
          value
        }
      }
    }
  }
}
```
