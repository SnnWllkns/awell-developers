---
title: Get checklist Details
description: Get the details of a checklist activity
---

This request allows us to fetch the details of a checklist activity.

## Request

### Query

```graphql
query GetChecklist($id: String!) {
  checklist(id: $id) {
    checklist {
      title
      items
    }
  }
}
```

### Variables

An example `id` for a checklist is "9rjfb9qE4pEY".

```json
{
  "id": "{{CHECKLIST_ID}}"
}
```

<br />

#### Where to find the checklist id

The id of the checklist can be found in the `object.id` field of an activity.

```json
{
  "activity": {
    "id": "{{ACTIVITY_ID}}",
    "object": {
      "id": "{{CHECKLIST_ID}}",
      "type": "checklist"
    },
    ...
  }
}
```
