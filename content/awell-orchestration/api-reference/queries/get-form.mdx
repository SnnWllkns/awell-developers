---
title: Get form
description: Get the form definition of a given form
---

This query allows us to fetch the details and questions for a specific form.

## Request

### Query

```graphql
query GetForm($id: String!) {
  form(id: $id) {
    form {
      id
      title
      questions {
        id
        title
        dataPointValueType
        questionType
        options {
          id
          value
          label
        }
        rule {
          # visibility conditions of a question
          boolean_operator
          conditions {
            id
            reference
            reference_key
            operator
            operand
          }
        }
      }
    }
  }
}
```

### Variables

```json
{
  "id": "{{FORM_ID}}"
}
```

<br />

#### Where to find the form id

The id of the form can be found in the `object.id` field of an activity.

```json
{
  "activity": {
    "id": "{{ACTIVITY_ID}}",
    "object": {
      "id": "{{FORM_ID}}",
      "type": "form"
    },
    ...
  }
}
```
