---
title: Get clinical note
description: Get the narratives and context of a clinical note
---

This query allows us to fetch the details of a clinical note.

## Request

### Query

```graphql
query GetClinicalNote($id: String!) {
  clinicalNote(id: $id) {
    clinical_note {
      id
      narratives {
        id # Our unique id for each narrative in a care flow
        key # A unique id you have set for your narrative
        title
        body # A string containing html content
      }
      context {
        key
        value
      }
    }
  }
}
```

### Variables

```json
{
  "id": "{{CLINICAL_NOTE_ID}}"
}
```

#### Where to found the clinical note id

The id of the clinical note can be found in the `object.id` field of an activity.

```json
{
  "activity": {
    "id": "{{ACTIVITY_ID}}",
    "object": {
      "id": "{{CLINICAL_NOTE_ID}}",
      "type": "clinical_note",
      "name": "Clinical note"
    },
    ...
  }
}
```
