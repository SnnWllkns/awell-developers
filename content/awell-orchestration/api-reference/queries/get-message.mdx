---
title: Get message
description: Get the body and attachments of a message activity
---

This request allows us to fetch the details of a message activity.

## Request

### Query

```graphql
query GetMessage($id: String!) {
  message(id: $id) {
    message {
      id
      body
      format
      subject
      attachments {
        id
        name
        url
        type
      }
    }
  }
}
```

### Variables

An example `id` for a message is "9rjfb9qE4pEY".

```json
{
  "id": "{{MESSAGE_ID}}"
}
```

<br />

#### Where to find the message id

The id of the message can be found in the `object.id` field of an activity.

```json
{
  "activity": {
    "id": "{{ACTIVITY_ID}}",
    "object": {
      "id": "{{MESSAGE_ID}}",
      "type": "message"
    },
    ...
  }
}
```
