---
title: Get all forms in a published pathway
description: Get all the form definitions in a published pathway
---

This query allows us to fetch the details and questions for all form defintions in a published pathway.

## Request

### Query

```graphql
query GetFormsInPublishedPathway(
  $pathway_definition_id: String!
  $release_id: String # release_id is optional: if not specified, we return forms for the latest published version of the pathway
) {
  forms(
    pathway_definition_id: $pathway_definition_id
    release_id: $release_id
  ) {
    forms {
      id
      title
      questions {
        id
        title
        dataPointValueType
        questionType
        options {
          id
          value
          label
        }
        rule { # visibility conditions of a question
          boolean_operator
          conditions {
            id
            reference
            reference_key
            operator
            operand
          }
        }
      }
    }
  }
}
```

### Variables

```json
{
  "pathway_definition_id": "{{PATHWAY_DEFINITION_ID}}",
  "release_id": "{{PATHWAY_RELEASE_ID}}"
}
```
