---
title: Create patient
description: Create a new patient, anonymous patients are also supported.
---

## Request

### Mutation

```graphql
mutation CreatePatient($input: CreatePatientInput!) {
  createPatient(input: $input) {
    patient {
      id
    }
  }
}
```

### Variables - Anonymous patient

```json
{
  "input": {}
}
```

### Variables - Patient with profile

Use our <a target="_blank" rel="noopener noreferrer" href="/playground">GraphQL explorer</a> to discover all the variables you can pass with this mutation.

```json
{
  "input": {
    "first_name": "{{PATIENT FIRST NAME}}",
    "last_name": "{{PATIENT LAST NAME}}",
    "patient_code": "{{YOUR PATIENT IDENTIFIER}}"
  }
}
```
