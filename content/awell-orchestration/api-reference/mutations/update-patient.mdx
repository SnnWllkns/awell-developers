## Request

### Mutation

```graphql
mutation UpdatePatient($input: UpdatePatientInput!) {
  updatePatient(input: $input) {
    patient {
      id
    }
  }
}
```

### Variables

Have a look at the validation section of the [createPatient mutation](/awell-orchestration/api-reference/mutations/create-patient#validation) to learn more about in what format values for certain variables should be passed.

```json
{
  "input": {
    "patient_id": "{{PATIENT_ID}}",
    "profile": {...}
  }
}
```
