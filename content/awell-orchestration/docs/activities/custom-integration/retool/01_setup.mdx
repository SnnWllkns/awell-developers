## Sign up

Sign up on the [Retool website](https://retool.com/) to get started.

## Add a resource

Retool allows us to build apps & internal tools where data can come from one or multiple resources (i.e. a database or API). In this case, data will be served by the Awell GraphQL API so lets create a GraphQL resource first.

**You will need the following information to configure your GraphQL resource:**

- API url: see [Endpoints](/awell-orchestration/api-reference/overview/endpoints)
- Headers (API key): see [Authorization](/awell-orchestration/api-reference/overview/authorization)

[Guide on how to create a resource in Retool](https://docs.retool.com/docs/resources).
