## Added

### GetClinicalNote query

GraphQL query to fetch clinical note for a given ID.

```graphql
query GetClinicalNote($id: String!) {
  clinicalNote(id: $id) {
    clinical_note {
      id
      narratives {
        id
        key
        title
        body
      }
      context {
        key
        value
      }
    }
  }
}
```
